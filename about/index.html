About me

Security researcher with a builder mindset

I focus on web application security and offensive testing workflows. My core strength is turning research output into practical tools, reusable scripts, and reports that make remediation straightforward.

What I deliver

  • Structured vulnerability validation for XSS, CSP bypass behavior, and message-channel abuse paths.
  • Automation scripts that reduce manual recon and speed up issue triage across multiple targets.
  • Clear technical writeups that connect exploit steps with risk and remediation guidance.
  • Prototype-ready tooling that teams can run quickly without heavy setup overhead.

How I work

  • Start with threat-oriented scope and map the highest-value attack surfaces first.
  • Use automation for collection and normalization, then switch to manual deep testing.
  • Capture proof-of-impact evidence early so reporting stays reproducible.
  • Keep workflows simple enough that they can be repeated under time pressure.
